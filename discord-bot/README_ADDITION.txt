### Setup
1. `npm install`
2. Copy `.env.example` to `.env` and fill in your values
3. `npm run delete` (to remove old commands)
4. `npm run deploy` (to deploy new commands)
5. `npm start` (to start the bot)

### Commands
- `npm start` - Start the bot
- `npm run deploy` - Deploy commands to Discord
- `npm run list` - List currently deployed commands
- `npm run delete` - Delete all deployed commands
- `npm run dev` - Start with auto-reload
